Quick start

  1. Upload the plugin to your Wordpress and enable it.
  2. Go to WooCommerce->Settings->Shipping->UPS, enter your UPS credentials and settings.
  3. Click the checkbox "Enable UPS" and Save the settings and voila!

Installation

 

IMPORTANT - ALWAYS BACKUP YOU SYSTEM BEFORE MAKING ANY UPGRADES OR CHANGES!

 

UPS Shipping is a Wordpress Plugin which allows the WooCommerce plugin to use UPS as a shipping provider.

Installation of this plugin is fairly simple. To install a WordPress Plugin automatically with the WordPress built-in installer, you need to manually upload this plugin to WordPress.

  1. Under your Wordpress admin view select "Plugins"
  2. From the submenus under the "Plugins" menu select "Add New"
  3. Select the "Upload" tab option from the links on the top of the "Install Plugins" page
  4. Press the "Browser" button to select the synmedia-woocommerce-shipping-ups.zip file and press "Install Now" button
  5. Once installed go to your Plugins page again and activate the plugin "WooMedia WooCommerce UPS"

Updates

 

To update the plugin, follow these directions:

  1. Delete the old plugin folder
  2. Upload the new folder
  3. Make sure to check if the extension is activated

Usages

 

Follow these steps to setup UPS Shipping:

  1. In your WooCommerce Settings, select Shipping then UPS.
  2. Enter the title of your UPS method, this title will be use to display the shipping rates.
  3. Enter your orgin postal code / zip code / post code.
  4. Enter your UPS access code, User ID, Password and Shipper Number.
  5. Select the Shipping methods you offer.

Troubleshooting

No rates are displayed

Verify that you have entered your Origin post code, Access number, User ID, Password and Shipper number. Then if no rates are displayed verify that your product contain the dimensions and weight.
